# BudgetNest

## Project Overview

BudgetNest is a Django-based web application developed to help families manage and track their daily expenses efficiently. The system allows authenticated users to add family members, record expenses, generate monthly and yearly reports, and analyze financial summaries.

The project is designed using Django framework with MySQL database integration and Bootstrap-based responsive frontend design.

---

# Features

## User Authentication
- User Signup
- User Login
- Secure Logout
- Session Management

## Family Management
- Add Family Members
- Update Family Member Details
- Delete Family Members
- View Family Data

## Expense Management
- Add Expenses
- Update Expenses
- Delete Expenses
- View Expense Records

## Financial Analytics
- Monthly Expense Reports
- Yearly Expense Reports
- Total Expense Calculation
- Financial Summary Dashboard

## Security Features
- CSRF Protection
- Login Authentication
- User-Specific Data Filtering
